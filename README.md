# market-intelligence-notebooklm
Caderno temático utilizando NotebookLM, IA generativa e dados públicos (IBGE/SIDRA e BACEN/SGS) para estudos em Inteligência de Mercado e análise estratégica.
# 📊 Inteligência de Mercado com IA e Dados Públicos

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da DIO com foco no uso do NotebookLM como ferramenta de aprendizagem ativa.

O objetivo principal foi estudar como dados públicos disponibilizados pelo Banco Central do Brasil (SGS) e pelo IBGE (SIDRA) podem ser utilizados em análises de Inteligência de Mercado, comportamento de consumo e tomada de decisão estratégica.

Além da pesquisa teórica, o projeto também explorou o uso de IA generativa para:
- síntese de conteúdos;
- estruturação de estudos;
- engenharia de prompts;
- criação de resumos e glossários;
- apoio na interpretação de dados econômicos e demográficos.

---

# 🎯 Objetivos de Estudo

- Compreender a estrutura do SGS (Banco Central) e SIDRA (IBGE);
- Aprender a localizar séries temporais e indicadores públicos;
- Explorar o uso de IA como ferramenta de apoio analítico;
- Desenvolver capacidade de interpretação estratégica de dados;
- Construir um material reutilizável para estudos futuros.

# 🧠 Engenharia de Prompts

## Prompt 1
"Explique como funciona a estrutura de tabelas do SIDRA para iniciantes."

### Resultado:
A IA conseguiu explicar os níveis geográficos e variáveis do SIDRA de forma simplificada.

### Dificuldade Encontrada:
As respostas iniciais estavam muito genéricas e sem exemplos práticos.

---

## Prompt 2
"Mostre exemplos práticos de uso do SGS em Inteligência de Mercado."

### Resultado:
Foram identificadas aplicações relacionadas a:
- inflação;
- crédito;
- comportamento de consumo;
- análise macroeconômica.

### Ajuste Realizado:
Foi necessário especificar:
"com exemplos aplicados ao varejo e marketplace".

---

## Prompt 3
"Crie um fluxo de análise utilizando SIDRA + SGS + Power BI."

### Resultado:
A IA estruturou um pipeline analítico com:
- coleta;
- tratamento;
- visualização;
- interpretação estratégica.

# ⚠️ Troubleshooting ("Cicatrizes")

Durante o desenvolvimento do projeto, algumas dificuldades encontradas foram:

- dificuldade inicial em entender a estrutura das APIs;
- excesso de informações técnicas nas documentações oficiais;
- respostas muito amplas da IA quando os prompts eram pouco específicos;
- necessidade de contextualizar melhor os objetivos analíticos.

Principais aprendizados:
- prompts específicos geram respostas melhores;
- IA funciona melhor como apoio analítico do que como resposta definitiva;
- documentação oficial continua sendo essencial.

# 📖 Miniguia de Estudos

## 🔍 Resumo do SGS

O SGS é o sistema de séries temporais do Banco Central do Brasil e disponibiliza indicadores econômicos como:
- SELIC;
- inflação;
- câmbio;
- crédito;
- inadimplência.

Esses dados podem ser utilizados para análises econômicas e estratégicas.

---

## 🔍 Resumo do SIDRA

O SIDRA é o sistema de recuperação automática de dados do IBGE.

Ele permite acessar:
- dados demográficos;
- renda;
- emprego;
- consumo;
- habitação;
- indicadores sociais.

---

# 🧩 Glossário

| Termo | Significado |
|---|---|
| SGS | Sistema Gerenciador de Séries Temporais |
| SIDRA | Sistema IBGE de Recuperação Automática |
| API | Interface para acesso automatizado de dados |
| Série Temporal | Dados organizados ao longo do tempo |
| Prompt | Instrução enviada para IA |
